# aws-nodejs-directupload
prueba de uso para direct upload en aws-nodejs

### Keys
agregar en development.env las keys como se encuentra en el ejemplo env.example

AWS_S3_BUCKET = xxx  
AWS_AccessKeyId = xxx  
AWS_SecretKey = xxx  
AWS_REGION = xxx  

### prueba

- npm install
- npm start

o desde vs code "F5". 

### Route

browser a localhost:3000/Account  
subir un archivo
